# Code of conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, focus on the work, assume good faith, and don't make anyone feel stupid for asking questions. Researchers come from a lot of different backgrounds — peer is for all of them.

## Enforcement

Report violations to `ke@pika.art`. Reports are confidential. Sustained or severe violations result in temporary or permanent removal from project spaces (issues, PRs, discussions, Discord, etc.).
